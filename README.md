Sales Dashboard - Power BI

Project Overview

This project presents a sales analysis dashboard developed in Power BI. The goal is to transform raw sales data into meaningful insights through interactive visualizations.

Key Metrics

- Total Sales: 2 Billion
- Sales by Category
- Sales by Segment
- Sales by Region
- Sales Trend by Year

Insights

- Furniture generated the highest sales volume among all categories.
- The Consumer segment represented the largest share of total sales.
- Sales showed a positive trend over the analyzed years.
- The West region achieved the strongest sales performance.

Tools Used

- Power BI
- Excel
- Data Visualization
- Data Analysis

Objective

The purpose of this project is to practice data analysis and business intelligence concepts by creating interactive dashboards that support data-driven decision making.
